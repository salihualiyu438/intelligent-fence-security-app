# Intelligent Fence Security System

An AI-powered security solution built with Django that intelligently monitors and protects fenced areas. This system leverages computer vision using YOLO to detect intrusions and enhance perimeter security.

---

## Features
- Real-time monitoring of fenced areas
- AI-based intrusion detection
- Django-powered web interface
- Alerts and notifications system
- Scalable and modular design
